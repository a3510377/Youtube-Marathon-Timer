# youtube-work-overtime

現在 `加班台` 越來越流行，斗內的時間基本都是由實況主自行加於計時器上，這造成遺漏/浪費時間等問題本系統將 `youtube` 的超級訊息和加入會員等直播訊息進行統整，自動加減時間減輕直播主的困擾。

## 安裝

目前該系統只有 `windows` 版本。如果你為其他系統請安裝 [nodejs >= 16](https://nodejs.org/zh-tw/download/) 並執行

```sh
npm i -g yarn
yarn install
yarn start build
yarn start start
```
